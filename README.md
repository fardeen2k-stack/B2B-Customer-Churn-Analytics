# 📊 B2B Customer Churn Analytics

An end-to-end Business Intelligence and Data Analytics project that simulates a real B2B SaaS company.

## 🚀 Tech Stack

- PostgreSQL
- SQL
- Python
- Pandas
- Faker
- SQLAlchemy
- Jupyter Notebook
- Power BI

---

## 📂 Project Architecture

```
Python
    ↓
Generate Synthetic Business Data
    ↓
PostgreSQL Database
    ↓
SQL Business Analysis
    ↓
Python EDA
    ↓
Power BI Dashboard
```

---

- Sales Representatives
